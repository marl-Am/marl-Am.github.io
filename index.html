E-commerce Website (Testing: Click For Website)

Driven by the desire to learn the web development stack called T3 (Next.js, TypeScript, Tailwind CSS, tRPC, Prisma), I created an ecommerce site stocked with my beloved items, including old games, textbooks, and novels. Instead of letting them go to waste, I wanted to provide them with a new home where they would be appreciated. This solo-developed platform utilizes Next.js, React, Prisma, Stripe, and Clerk to ensure a seamless shopping experience. With responsive design, secure payments, and user management functionalities, the site is deployed on Vercel for scalability and optimal performance.

Music Recommendation Application (Click For Website)

A music recommendation app that uses a User's favorite artist to generate similar musicians. The motivation behind this project was my personal experience of discovering a new band and wanting to find more like them. React makes an api call to the Flask server that uses Spotipy and the Spotify Web API to find recommended artists. Styled using HTML/CSS/JavaScript/Tailwind to render the frontend. Deployed using Google Cloud's Cloud Run for the back-end and Firebase Hosting for the front-end.
